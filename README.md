# Prashrey Art Gallery

Futuristic digital art portfolio built with React, TypeScript, and Tailwind CSS.

## Project structure

```
src/
├── components/prashrey-art-gallery/
│   ├── PrashreyArtGallery.tsx   # Page layout (composes sections)
│   ├── BackgroundEffects.tsx
│   ├── Navbar.tsx
│   ├── HeroSection.tsx
│   ├── FeaturedStats.tsx
│   ├── GallerySection.tsx
│   ├── GalleryCard.tsx
│   ├── AboutSection.tsx
│   ├── ContactSection.tsx
│   └── Footer.tsx
├── data/
│   ├── artworks.ts
│   ├── navigation.ts
│   └── stats.ts
└── types/
    └── artwork.ts
```

## Development

```bash
npm install
npm run dev
```
