# Prashrey Palette Art Studio

A premium, responsive portfolio website for **Prashrey Palette Art Studio** — built with React, TypeScript, Tailwind CSS, Vite, and Framer Motion.

## Features

- Immersive hero with cinematic motion
- Animated hollow polygon background (scroll-reactive parallax)
- Filterable masonry gallery with artwork detail modal
- Horizontally scrollable featured collections
- About, testimonials, and contact sections
- Sticky transparent navbar with mobile menu

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
npm run preview
```

## Project Structure

```
src/
  components/       # UI components (Navbar, Hero, Gallery, etc.)
  data/             # Artworks, collections, testimonials, navigation
  hooks/            # useScrollProgress, useInView
  types/            # TypeScript types
  utils/            # Helpers (smooth scroll)
```
